Compatibility Issues:

I've tested CipherClerk's Applet with the following browsers: Netscape Namvigator 4.7, Microsoft Internet Explorer 5.0, and Opera 4.02. Additionally, I've tested it using Sun's Appletviewer in JDK 1.3. Finally, I testet the code as Java Application using Microsoft's Java VM and the VM's from Sun's JDK 1.3 and 1.1.8. All these test where done on Microsoft Windows.
Unfortunately I neither own different harware nor do I have a Linux / Unix box handy. The rest should be a matter of compatibility amonge the Java Virtual Machines. So while CipherClerk's Applet should run everywhere, but you might experiance unexpected behaviour, too.

Bugs & Changes:

Version 1.3b2
  • Fixed a bug that caused CipherClerk's Applet to crash within the Opera  browser whenever the page it's been loaded from was re-visited.
  • As I use CipherClerk as my nom-de-chiffre, the applet is now refered as CipherClerk's Applet.
Version 1.2b1
  • A second - reduced version distilled. Contains only a few ciphers, takes only half space. To be uses only if I run out of space on my home page.
  • For the full version, the selection of the cipher to be used was split into two parts.
  • Some ciphers added: Gronsfeld, GROMARK, T52.
  • Fixed a zillion typos, but left enough if you enjoy searching....
  • some other minor changes.
Version 1.1b2
  • More cipher machines added: Hagelin C-35, C-38; Kryha; Hebern; RED & PURPLE machine.
  • Simple Substitution: ability to mix both alphabets.
  • Help message added.
Version 1.1b1
  • User Interface modified: Cipher and operation are now chosen at the text input.
  • A simulation of the B211 cipher machine added.
Version 1.0b13
  • Some minor changes.
Version 1.0b11
  • Some minor GUI changes
  • The input text is optionally copied to the result.
  • Output setup is now conserved while the applet is running.
  • Nihilist substitution may now use 0-4 or 1-5 as row & column labels.
  • Polyalphabetic Substitution, Nihilist substitution, Bifid and Two square: Smart use of the second (and third) key word: The clear button now turns off the use of the key word. When entering these key words, pressing the enter key will turn on the use of the key word if it is acceptable.
  • CipherClerk didn't load from aprox. 15 to 29 August.
Version 1.0b10
  • Morse Code Based Ciphers added: Pollux, Morbit, Fractionated Morse
  • Added plain text/cipher text based interruption to polyalphabetic substitution
  • Choice of complete / irregular transposition changed to radio buttons.
Version 1.0b9
  • Introducing a new 36 letter alphabet.
  • Nihilist substitution completed.
Version 1.0b8
  • Nihilist substitution added. Note that this cipher is not implemented and described completely: So far, only the use of checkerboard is done. The final addition using a second key will be done soon, hopefully!
  • Append Cipher Info is now on by default.
  • ACA Homophonic cipher added
  • Whetstone cipher added
  • Redfence cipher added
  • Different key words for plain text and cipher text alphabets for polyalphabetic substitution cipher.
Version 1.0b7
